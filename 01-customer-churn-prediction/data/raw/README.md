# Raw data

**Dataset:** [Churn for Bank Customers (Kaggle)](https://www.kaggle.com/datasets/mathchi/churn-for-bank-customers)

1. Download the dataset from Kaggle (e.g. `Churn_Modelling.csv`).
2. Place the CSV in this folder.
3. Notebooks expect: `../data/raw/Churn_Modelling.csv` (adjust in notebooks if your filename differs).

Do not commit large CSV files if not needed; add `*.csv` to `.gitignore` and document the download link in the main README.
