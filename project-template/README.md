# Project Template

Use this structure when adding new portfolio projects. See [main README](../README.md) and [PORTFOLIO_OVERVIEW](../PORTFOLIO_OVERVIEW.md) for standards.

## Structure

```
project-name/
├── README.md
├── data/raw/
├── data/processed/
├── scripts/
├── sql/
├── visualizations/
├── reports/
├── requirements.txt
└── .gitignore
```
