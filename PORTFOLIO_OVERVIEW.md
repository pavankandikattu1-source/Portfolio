## Repository Structure (Per Project)

```
project-name/
├── README.md
├── data/raw/          # Original dataset or link
├── data/processed/    # Cleaned data
├── notebooks/         # 01_eda, 02_cleaning, 03_analysis
├── sql/               # queries.sql, schema (optional)
├── visualizations/    # dashboard.pbix, key_charts.png
├── reports/           # analysis_report.md, business_recommendations.md
├── requirements.txt
└── .gitignore
```
